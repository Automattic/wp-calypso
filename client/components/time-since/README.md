# Time Since

Shows a date relative to now in a human friendly format.
Updates the display as time progresses.

## Examples

Under a minute:
`Just now`

Under a week:
`20h ago`
`5d ago`

Over a week:
`Oct 30, 2017`

## Props

- `date`: Date, The date to show
- `dateFormat`: String, Override the formatting of dates older than 1 week.

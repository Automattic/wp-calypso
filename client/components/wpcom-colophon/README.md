WpcomColophon (JSX)
====================

This component is used to display a "In partnership with WordPress.com" colophon, usually on the page footer.

-------

#### How to use:

```js
import WpcomColophon from 'components/wpcom-colophon';

function MyFooter() {
	return (
		<div>
			<WpcomColophon />
		</div>
	);
}
```

# JetpackLegacyCard

This component is used to display a Jetpack legacy card.

---

#### How to use:

```js
import JetpackLegacyCard from 'components/jetpack/card/jetpack-legacy-card';

export default function JetpackLegacyCardExample() {
	return (
		<div>
			<JetpackLegacyCard { ...props } />
		</div>
	);
}
```

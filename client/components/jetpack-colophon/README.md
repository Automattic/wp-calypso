JetpackColophon (JSX)
====================

This component is used to display a "Powered by Jetpack" colophon, usually on the footer of page

-------

#### How to use:

```js
import JetpackColophon from 'components/jetpack-colophon';

function MyFooter() {
	return (
		<div>
			<JetpackColophon />
		</div>
	);
}
```
